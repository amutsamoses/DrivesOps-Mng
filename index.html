Overview

Welcome to the API documentation for the Vehicle Rental Management System. This API is built to provide backend services for managing users, vehicles, bookings, and more. The base URL is: https://drivesops-mng.onrender.com.

Ensure all requests are authenticated where necessary and adhere to the API guidelines for successful interactions.
