Welcome to the API Documentation

This API provides a variety of endpoints to interact with our application. Below is a list of available endpoints and their descriptions.

Authentication

POST /api/login

Authenticates a user and returns a JWT token.

Users

GET /users

Fetches a list of users.

POST /users

Creates a new user.

Branches

GET /branches

Fetches a list of branches.

POST /branches

Creates a new branches.

Fleet-Management

GET /fleet-management

Fetches a list of fleet-management.

POST /fleet-management

Creates a new fleet-management.

vehicle-specifications

GET /vehicle-specifications

Fetches a list of vehicle-specifications.

POST /vehicle-specifications

Creates a new vehicle-specifications.

Vehicles

GET /vehicles

Fetches a list of vehicles.

POST /vehicles

Creates a new vehicles.

support-tickets

GET /support-tickets

Fetches a list of support-tickets.

POST /support-tickets

Creates a new support-tickets.

Bookings

GET /bookings

Fetches a list of bookings.

POST /bookings

Creates a new bookings.

Stripe Checkout

POST /create-checkout-session

Creates a create-checkout-session.
